Initial release.

- Automated batch stitching for Keyence BZ-X800 microscope images
- Keyence Composite Batch Mode for continuous runs
- Automatic channel management (unchecks CH1/CH2/CH3, checks Overlay)
- Direct composite RGB export from Keyence Analyzer
- Advanced color detection for checkbox state verification
